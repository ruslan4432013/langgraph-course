# LangGraph Course

Этот проект является частью обучающего курса по LangGraph. В нем демонстрируется создание простого графа состояний с
использованием инструментов (tools).

![LangGraph Logo](static/img.png)

## Описание

Проект использует `langchain` и `langgraph` для построения агентов и рабочих процессов. В текущем примере (
`src/main.py`) реализован граф, который:

1. Принимает сообщения от пользователя.
2. Использует LLM (OpenAI) для определения необходимости вызова инструмента.
3. Содержит пример простого инструмента `multiply` для перемножения чисел.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Конфигурация

Создайте файл `.env` в корневом каталоге на основе `.env.example` и заполните необходимые переменные окружения:

- `LANGSMITH_API_KEY` - ключ для LangSmith (опционально для трассировки).
- `OPENAI_API_KEY` - ваш API ключ OpenAI.

## Запуск

Для запуска примера выполните:

```bash
python src/main.py
```

## Структура проекта

- `src/main.py` - основной код с определением графа и логикой LLM.
- `src/settings.py` - настройки приложения и загрузка переменных окружения.
- `langgraph.json` - конфигурация для LangGraph CLI.
- `static/` - статические файлы (изображения).
