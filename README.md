# LangGraph Course

Учебный проект по изучению библиотеки **LangGraph**. В репозитории демонстрируются основные концепции построения графов,
работы с состоянием (State) и реализации логики переходов.

## О проекте

Проект содержит примеры реализации AI-агентов и рабочих процессов (workflows) с использованием LangGraph. В текущей
версии реализован демонстрационный граф, который показывает:

- Инициализацию состояния графа.
- Работу узлов (Nodes) как Python-функций.
- Использование условных рёбер (Conditional Edges) для ветвления логики.
- Процесс компиляции и запуска графа.

## Структура проекта

- `src/main.py` — основной файл с примером простого графа.
- `langgraph.json` — конфигурация для LangGraph CLI/Studio.
- `requirements.txt` — список необходимых зависимостей.
- `.env.example` — пример файла с переменными окружения (API ключи).

## Установка

1. Клонируйте репозиторий.
2. Создайте виртуальное окружение:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   # или
   .venv\Scripts\activate     # Для Windows
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Настройте переменные окружения (опционально, для работы с LLM):
   ```bash
   cp .env.example .env
   # Отредактируйте .env, добавив свои ключи
   ```

## Запуск демонстрации

Для запуска примера простого графа выполните:

```bash
python src/main.py
```

### Схема работы демонстрационного графа

![Схема графа](static/img.png)

Граф имитирует простую логику формирования предложения:

1. **START** → `node_1`: Добавляет к строке фразу "I am".
2. **node_1** → `decide_mood`: Случайным образом выбирает путь (50/50).
3. **node_2** (Happy): Добавляет "happy!".
4. **node_3** (Sad): Добавляет "sad!".
5. **END**: Завершение работы.

## Использование с LangGraph Studio

Проект подготовлен для работы с LangGraph Studio. Файл `langgraph.json` указывает на точку входа `src/main.py:graph`.
