# Примеры кода по теме Векторные хранилища

## Описание примеров:
1. Инициализация — создание экземпляра InMemoryVectorStore с моделью эмбеддингов.

2. Добавление документов — создание Document объектов с page_content и metadata, добавление их в хранилище с явными ID.

3. Поиск по сходству — выполнение семантического поиска с использованием метода similarity_search().

4. Удаление документов — удаление документов из хранилища по ID и проверка результатов.

5. Фильтрация по метаданным — демонстрация концепции фильтрации с пояснением возможностей различных хранилищ.

6. Метрики сходства — реализация и демонстрация трёх основных метрик (косинусное сходство, евклидово расстояние, скалярное произведение).

7. Структура Document — показ внутреннего устройства Document объектов с примерами различных метаданных.

8. Параметры поиска — демонстрация влияния параметра k на количество возвращаемых результатов.

9. Полный рабочий цикл — комплексный пример, охватывающий инициализацию, добавление, поиск и удаление.

10. Продвинутые концепции — информация о MMR, гибридном поиске, индексировании и рекомендациях по выбору хранилища.

Все примеры готовы к запуску без дополнительных импортов, используют встроенные эмбеддинги для демонстрации и содержат подробные комментарии на русском языке
